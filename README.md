# Algorithm and Data Structures Camp 🚀

Welcome to my repository dedicated to mastering fundamental algorithms and data structures using C#! This project serves as a comprehensive archive of the problem-solving patterns and efficiency practices I developed during my technical training.

## 📌 Project Purpose
The goal of this camp was to build a strong theoretical foundation in computer engineering logic, focusing on code efficiency, space-time complexity (Big O notation), and performance optimization.

## 🛠️ Technologies & Tools
* **Language:** C# (.NET Core)
* **IDE:** Visual Studio
* **Version Control:** Git & GitHub

## 📂 Repository Structure
Inside this repository, you will find organized folders containing:
* Core Data Structures (Arrays, Lists, Custom Collections)
* Sorting and Searching Algorithms
* Mathematical and Logical Problem-Solving Scripts

## 🚀 How to Run
1. Clone this repository to your local machine.
2. Open the solution file using Visual Studio.
3. Select the specific algorithm project you want to examine and click **Start/Play**.
